Имя метода: ПроверитьРавенствоДатСТочностью2Секунды

Синтаксис:

ПроверитьРавенствоДатСТочностью2Секунды(ПерваяДата, ВтораяДата, ДопСообщениеОшибки = "")

Описание:

Проверяет равенство дат с точностью до двух секунд.

Тип возвращаемого значения: 

Описание возвращаемого значения: 

Структура - Если Vanessa-Automation в режиме ожидания продолжения шагов возвращает СтруктураОшибкаАссерта.

Параметры:
  • ПерваяДата  (Обязательный).

Тип: Дата.

Описание: Первая проверяемая дата.

  • ВтораяДата  (Обязательный).

Тип: Дата.

Описание: Вторая проверяемая дата.

  • ДопСообщениеОшибки  (Необязательный).

Тип: Строка.

Описание: Описание ошибки, которое будет использовано в тексте исключения ассерта.

Доступность:

  • Сервер
  • Тонкий клиент
  • Толстый клиент (управляемое приложение)
  • Толстый клиент (обычное приложение)

Пример:

Результат = Обработки.ОписаниеАссертов.ПроверитьРавенствоДатСТочностью2Секунды(ПерваяДата, ВтораяДата, ДопСообщениеОшибки)
