Имя метода: ПроверитьВалидностьЗапросаСКД

Синтаксис:

ПроверитьВалидностьЗапросаСКД(ТекстЗапроса, ДопСообщениеОшибки = "")

Описание:

Проверяет, что запрос СКД не вызовет ошибок компиляции.

Параметры:
  • ТекстЗапроса  (Обязательный).

Тип: Строка.

Описание: Текст запроса.

  • ДопСообщениеОшибки  (Необязательный).

Тип: Строка.

Описание: Описание ошибки, которое будет использовано в тексте исключения ассерта.

Доступность:

  • Сервер
  • Тонкий клиент
  • Толстый клиент (управляемое приложение)
  • Толстый клиент (обычное приложение)

Пример:

Обработки.ОписаниеАссертов.ПроверитьВалидностьЗапросаСКД(ТекстЗапроса, ДопСообщениеОшибки)
