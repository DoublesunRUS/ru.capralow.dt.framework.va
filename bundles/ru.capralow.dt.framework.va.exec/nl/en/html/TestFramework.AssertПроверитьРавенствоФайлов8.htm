Имя метода: AssertПроверитьРавенствоФайлов

Синтаксис:

Процедура AssertПроверитьРавенствоФайлов(ПутьФайла1, ПутьФайла2, ДопСообщениеОшибки = "", ПарамСравнениеФайлов = Неопределено, ПоказатьРазличия = Ложь) Экспорт

Описание:

Проверяет равенство файлов

Параметры:
  • ПутьФайла1, ПутьФайла2  (Обязательный).

Тип: Булево.

Описание: полные пути к файлам

  • ДопСообщениеОшибки  (Обязательный).

Тип: Булево.

Описание: Описание ошибки, которое будет использовано в тексте исключения ассерта.

  • ПарамСравнениеФайлов  (Обязательный).

Тип: Булево.

Описание: структура с ключами:

  • * ИгнорироватьПустоеПространство  (Обязательный).

Тип: Булево.

Описание:

  • * СпособСравнения  (Обязательный).

Тип: Булево.

Описание:

  • * УчитыватьРазделителиСтрок  (Обязательный).

Тип: Булево.

Описание:

  • * УчитыватьРегистр  (Обязательный).

Тип: Булево.

Описание:

  • ПоказатьРазличия  (Обязательный).

Тип: Булево.

Описание: в случае ошибки появляется окно сравнения (для отладки)

Доступность:

  • Сервер
  • Тонкий клиент
  • Толстый клиент (управляемое приложение)
  • Толстый клиент (обычное приложение)

Пример:

Обработки.ОписаниеАссертов.AssertПроверитьРавенствоФайлов(ПутьФайла1, ПутьФайла2, ДопСообщениеОшибки, ПарамСравнениеФайлов, ПоказатьРазличия)
