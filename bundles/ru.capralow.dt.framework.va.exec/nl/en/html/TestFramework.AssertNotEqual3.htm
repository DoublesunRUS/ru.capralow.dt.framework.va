Имя метода: AssertNotEqual

Синтаксис:

Функция AssertNotEqual(NotExpected, Actual, Message = "") Экспорт

Описание:

Проверяет неравенство двух значений.

Тип возвращаемого значения: 

Описание возвращаемого значения: 

Структура - Если Vanessa-Automation в режиме ожидания продолжения шагов возвращает СтруктураОшибкаАссерта.

Параметры:
  • ПервоеЗначение  (Обязательный).

Тип: Булево.

Описание: Первое проверяемое значение.

  • ВтороеЗначение  (Обязательный).

Тип: Булево.

Описание: Второе проверяемое значение.

  • ДопСообщениеОшибки  (Необязательный).

Тип: Булево.

Описание: Описание ошибки, которое будет использовано в тексте исключения ассерта.

Доступность:

  • Сервер
  • Тонкий клиент
  • Толстый клиент (управляемое приложение)
  • Толстый клиент (обычное приложение)

Пример:

Результат = Обработки.ОписаниеАссертов.AssertNotEqual(NotExpected, Actual, Message)
