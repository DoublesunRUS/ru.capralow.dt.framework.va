Имя метода: ПроверитьРавенствоТаблиц

Синтаксис:

ПроверитьРавенствоТаблиц(Таб1, Таб2, ДопСообщениеОшибки = "", ДопПараметры = Неопределено)

Описание:

Проверяет, что таблицы равны.

Параметры:
  • Таб1  (Обязательный).

Тип: ТабличныйДокумент.

Описание: Коллекция значений.

  • Таб2  (Обязательный).

Тип: ТабличныйДокумент.

Описание: Коллекция значений.

  • ДопСообщениеОшибки  (Необязательный).

Тип: Строка.

Описание: Описание ошибки, которое будет использовано в тексте исключения ассерта.

  • ДопПараметры  (Необязательный).

Тип: Структура.

Описание: Дополнительные параметры.

Доступность:

  • Сервер
  • Тонкий клиент
  • Толстый клиент (управляемое приложение)
  • Толстый клиент (обычное приложение)

Пример:

Обработки.ОписаниеАссертов.ПроверитьРавенствоТаблиц(Таб1, Таб2, ДопСообщениеОшибки, ДопПараметры)
